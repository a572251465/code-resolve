## zh-CN

通过 `TimePicker.RangePicker` 使用时间范围选择器。

## en-US

Use time range picker with `TimePicker.RangePicker`.

```tsx
import React from 'react';
import { TimePicker } from 'antd';

const App: React.FC = () => <TimePicker.RangePicker />;

export default App;
```
